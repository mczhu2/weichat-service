## 二次验证注意事项以及操作流程

# 首次登陆账号可能会出现二次验证的问题 30分钟内出现 尽量上号等30分钟
  ![](https://www.showdoc.com.cn/server/api/attachment/visitFile?sign=fcada2801ed3688a141ecabd621381a8&file=file.png)
  1. 出现以上提示会回调一个二次验证通知，在下发-》登录-》[二次验证回调](https://www.showdoc.com.cn/1889765213630885/10821668541626724)
  2. 请求接口的时候也会提示需要二次验证，如下
```
	{
		"data":null,
		"errcode":-11025,
		"errmsg":"出现二次验证，请重新获取二维码登录"
	}
```
3. 根据以上提示调用获取二次验证二维码接口 调用-》登录-》[获取二次验证二维码](https://www.showdoc.com.cn/1889765213630885/10821789051643255)


4. 然后点击下图进行扫码验证
![](https://www.showdoc.com.cn/server/api/attachment/visitFile?sign=1f8883e44b73606e21cc1fd4b18ebb79&file=file.png)
5. 扫码后会出现下图提示点击确定就可以解决了
![](https://www.showdoc.com.cn/server/api/attachment/visitFile?sign=3413370296aa6aa32de353916a3a2d7e&file=file.png)

## 自动登录重连流程


# 首次登录流程

1. [初始化接口调用](https://www.showdoc.com.cn/1889765213630885/8704551724545128 "初始化接口调用")
![](https://www.showdoc.com.cn/server/api/attachment/visitFile?sign=2b42517732a732411ebc5c36d3f7437f&file=file.png)
2. 设置回调地址接口，没有忽略
3. [获取登录二维码](https://www.showdoc.com.cn/1889765213630885/8818496881615449 "获取登录二维码")
4. [输入验证码接口](https://www.showdoc.com.cn/1889765213630885/10480302757322729 "输入验证码接口")  新设备第一次必现验证码


# 自动登录流程
##### 如果首次登录了，然后调用我的接口退出了，或者我把服务给关了重启了一下，这时候重新初始化，就可以调用自动登录登陆上。

1. 调用[初始化接口调用](https://www.showdoc.com.cn/1889765213630885/8704551724545128 "初始化接口调用")  参数vid 填首次登录的账号vid 1688开头的。
![](https://www.showdoc.com.cn/server/api/attachment/visitFile?sign=284c7f349f07a470a55e95388e3115b5&file=file.png)
2. 调用[自动登录接口](https://www.showdoc.com.cn/1889765213630885/8818497363410458 "自动登录接口")  返回成功就代表登录上了，返回失败就去调用 [获取登录二维码](https://www.showdoc.com.cn/1889765213630885/8818496881615449 "获取登录二维码") 不用重新初始化，直接获取登录二维码，扫码登录，这时候初始化接口vid传入了值，就会默认读取首次登陆的设备信息，就不会提示新设备，扫完就能登陆成功。

# 注意事项

## 遇到重复消息原因。

### 1. 细节说明  
- **具体内容**: 遇到同一个消息回调多次的情况，请看json中的referid字段如果是0就是原消息，不是0就是一个在原本消息上操作的回调，比如已读操作。

## 文件下载说明。

### 1. 细节说明  
- **具体内容**: 
1. 文件消息有两种，一个是外部联系人发送的文件消息，比如个微发送的图片 视频 文件，走外部联系人下载接口。
2. 如果是外部的企业好友，或者内部联系人发送的图片 视频 文件 语音走cdn文件下载接口。
