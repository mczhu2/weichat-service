## 二次验证消息通知



    
**简要描述：** 

- 二次验证消息通知

**返回类型：** 
- ` 100012 `

**返回uuid：** 
- ` f7503bb5-7d27-408f-ab24-8c4ace7f068e `

**返回示例**
``` 
{
    "vid":1688856591445501,
    "msg":"出现二次验证，请重新获取二维码扫码验证登录",
    "uuid":"680554c2b347acf9b77b73ac72448afa"
}
```
